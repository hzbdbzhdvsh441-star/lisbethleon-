Proyecto de Instalación de Red LAN

Unidad Educativa Buena Fe – Aula de Sistemas Informáticos

Descripción del Proyecto

Este proyecto consiste en la instalación y configuración de una red LAN en el aula de Sistemas Informáticos de la Unidad Educativa Buena Fe. La red permitirá conectar todas las computadoras entre sí, compartir recursos y acceder a internet de forma estable para apoyar el aprendizaje de los estudiantes.

Objetivo General

Implementar una red LAN funcional que permita el trabajo colaborativo, el acceso a herramientas digitales y la conexión eficiente a internet dentro del aula.

Componentes del Sistema

Componente	Cantidad	Descripción

Router	1	Distribuye la conexión a internet
Switch	1	Conecta todas las computadoras entre sí
Computadoras	Según aula	Equipos de estudiantes
Cable UTP CAT6	Variable	Cableado principal de red
Conectores RJ45	Variable	Terminales de conexión
Canaletas	Variable	Organización del cableado
Tester de red	1	Comprobación de funcionamiento de cables


Arquitectura de la Red

La red utiliza una topología estrella, donde el switch es el punto central de conexión.

Internet → Módem → Router → Switch → Computadoras

Procedimiento de Instalación

1. Medición y planificación del recorrido de cableado.


2. Colocación de canaletas en el aula.


3. Corte y armado de cables UTP con conectores RJ45.


4. Instalación física del switch y router.


5. Conexión de todas las computadoras.


6. Configuración de red (direcciones IP, DHCP, pruebas de conexión).


7. Verificación del funcionamiento general.



Configuración

Asignación automática de IP (DHCP activado en el router).

Seguridad del router configurada con contraseña.

WiFi (opcional) solo para administración del sistema.


Resultados y Beneficios

Conectividad estable entre todas las computadoras.

Acceso a internet para actividades académicas.

Facilidad para compartir archivos y herramientas educativas.

Mejora en la calidad del aprendizaje y prácticas tecnológicas.


Conclusión

La instalación de la red LAN permitió modernizar el aula de Sistemas Informáticos, mejorando la interacción, el trabajo colaborativo y el acceso a recursos digitales dentro de la Unidad Educativa Buena Fe.
